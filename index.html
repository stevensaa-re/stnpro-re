Tentang STN PROJECT REBORN

STN PROJECT REBORN adalah platform yang saya buat untuk berbagi passion di berbagai bidang. Dari menangani sound system untuk event, fotografi profesional, proyek elektronika custom, web development, dan konsultasi seputar rakitan komputer.
